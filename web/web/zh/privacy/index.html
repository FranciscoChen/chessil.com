隐私政策

本政策说明 ChessiL.com 在你使用我们的国际象棋服务时如何收集、使用并保护个人数据。

概览

我们收集的数据

账户信息、对局数据以及提供服务所需的基本技术数据。

如何使用

用于运行对局、保护账户安全、检测作弊并提升可靠性。

你的控制

你可以访问或删除账户数据，并通过站点请求帮助。

收集的信息

  • 账户数据：用户名和密码（使用 argon2id 哈希存储）。
  • 对局数据：对局、走子和分析结果。
  • 技术数据：IP 地址、会话标识符和 HTTP 请求元数据。
  • 认证与会话数据，用于保持登录状态。

信息用途

  • 提供实时对局、分析和学习工具。
  • 验证用户身份并保护账户安全。
  • 检测滥用、作弊和欺诈。
  • 维护系统可靠性与性能。

处理的法律依据（GDPR）

  • 合同：为你提供所请求的服务。
  • 合法利益：保障平台安全并防止滥用。
  • 同意：在法律要求时，例如可选 cookies。
  • 法律义务：如需配合法律要求。

数据保留

账户及相关数据可能在一年不活跃后删除。IP 地址和请求日志保留最长一年以防止滥用并维护安全。会话 Cookie 在一周不活跃或一年后过期，以先到者为准。账户删除后对局数据将匿名化，若双方均匿名则会被删除。

共享与传输

我们不会出售你的数据。我们可能与帮助托管、安全或运营平台的服务提供商共享数据，并受保密义务约束。 如涉及跨境处理，我们会采取适当的保护措施。

安全

我们使用行业标准安全实践，包括密码哈希、安全会话处理和滥用监测。任何在线系统都无法做到 100% 安全， 但我们持续努力保护你的数据。

你的权利

  • 访问、更正或删除你的账户数据。
  • 在法律适用范围内反对或限制某些处理。
  • 请求以可移植格式获取你的数据副本。

未成年人

ChessiL.com 面向未成年人也应安全使用。如你未达到需要家长同意的年龄，必须取得父母或监护人的许可。

政策变更

我们可能不时更新本政策，并在本页面发布新版本及更新后的生效日期。
