Términos del servicio

ChessiL.com es un sitio web de ajedrez.

Estos términos del servicio son un acuerdo legalmente vinculante.

ChessiL.com ofrece un sitio web y servicio de ajedrez en línea.

Al acceder o usar el sitio web o los servicios, aceptas estos términos:

Descargo de responsabilidad

El sitio web y los servicios se proporcionan "tal cual"; ChessiL.com no será responsable de ningún daño o pérdida de cualquier tipo derivados del uso del sitio web o los servicios.

Uso del sitio web o los servicios

No accederás sin autorización a ningún computador, sistema, servidor, red, datos, cuenta o base de datos conectados al sitio web o los servicios.

No atacarás el sitio web o los servicios con ataques volumétricos o de denegación de servicio.

No explotarás errores o vulnerabilidades del sitio web o los servicios.

Crear una cuenta

Puedes crear una cuenta para jugar partidas clasificadas.

No crearás más de 4 cuentas.

Eres responsable de toda la actividad que ocurra en tu cuenta.

No compartirás tu cuenta ni tu contraseña con nadie.

No suplantarás a nadie.

Elegirás un nombre de usuario adecuado para niños y que no ofenda ni dañe a nadie.

Las cuentas deben usarse; por lo tanto, ChessiL.com eliminará tu cuenta después de 1 año de inactividad.

Juego limpio

No recibirás ayuda mientras juegas.

Jugarás honestamente y tratarás de ganar.

No arreglarás resultados con nadie.

No jugarás contra ti mismo para subir tu puntuación.

No abandonarás tus partidas obligando a tus oponentes a esperarte.

No abortarás demasiadas partidas.

ChessiL.com se reserva el derecho de eliminar tu cuenta o tomar cualquier otra acción por cualquier razón sin previo aviso.
