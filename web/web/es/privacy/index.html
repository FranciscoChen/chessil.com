Política de privacidad

ChessiL.com es un sitio web de ajedrez.

ChessiL.com recopilará y usará solo los datos necesarios para prestar sus servicios.

Puedes crear una cuenta para jugar partidas clasificadas y guardar un historial de partidas.

Las cuentas y todos los datos relacionados se eliminarán después de 1 año de inactividad.

Tus datos no se compartirán con terceros.

Dirección IP

Obligatoria para establecer una conexión a internet.

Se usa para prevenir ataques cibernéticos y fraudes.

Se elimina después de 1 año.

Cookie de sesión

ChessiL.com te la entrega; mantenla segura y en secreto.

Se usa para autenticarte y mantener tu sesión iniciada.

Se usa para jugar partidas y mover como tú.

Se usa para identificar usuarios no registrados.

Se elimina tras 1 semana de inactividad o 1 año.

Solicitudes HTTP

Enviadas a ChessiL.com por ti.

Se usan para navegar el sitio web.

Se usan para prevenir ataques cibernéticos y fraudes.

Se eliminan después de 1 año.

Nombre de usuario

Proporcionado voluntariamente a ChessiL.com para crear una cuenta.

Se usa para identificarte ante ChessiL.com y otros usuarios.

Se elimina junto con la eliminación de la cuenta.

Contraseña

Proporcionada voluntariamente a ChessiL.com para crear una cuenta.

Se cifra con argon2id y nunca se guarda en texto plano.

Se usa para iniciar sesión en tu cuenta.

Se elimina junto con la eliminación de la cuenta.

Partidas y movimientos

Proporcionados voluntariamente a ChessiL.com al jugar.

Se usan para mostrar tus partidas anteriores y para prevenir trampas.

Se anonimizan junto con la eliminación de la cuenta.

Se eliminan cuando ambos jugadores son anónimos.
