Política de privacidad

ChessiL.com es un sitio web de ajedrez.

ChessiL.com recopilará y utilizará únicamente los datos necesarios para prestar sus servicios.

Puede crear una cuenta para jugar partidas clasificatorias y mantener un historial de partidas.

Las cuentas y todos los datos relacionados se eliminarán después de 1 año de inactividad.

Sus datos no serán compartidos con terceros.

Dirección IP

Obligatorio para establecer una conexión a Internet.

Se utiliza para prevenir ataques cibernéticos y fraudes.

Eliminado después de 1 año.

Cookie de sesión

Proporcionado por ChessiL.com, manténgalo en secreto y a salvo.

Se utiliza para autenticarle y mantener su sesión iniciada.

Se utiliza para jugar partidas de ajedrez y para hacer sus movimientos.

Se utiliza para identificar usuarios no registrados.

Eliminado después de 1 semana de inactividad o 1 año.

Solicitudes HTTP

Entregado a ChessiL.com por usted.

Se utilizan para navegar por el sitio web.

Se utilizan para prevenir ataques cibernéticos y fraudes.

Eliminados después de 1 año.

Nombre de usuario

Entregado voluntariamente a ChessiL.com por usted para crear una cuenta.

Se utiliza para identificarse con ChessiL.com y otros usuarios.

Eliminado junto con la eliminación de la cuenta.

Contraseña

Entregado voluntariamente a ChessiL.com por usted para crear una cuenta.

Hash con argon2id y nunca almacenado en texto plano.

Se utiliza para iniciar sesión en su cuenta.

Eliminado junto con la eliminación de la cuenta.

Partidas y movimientos de ajedrez

Entregado voluntariamente a ChessiL.com por usted al jugar.

Se utilizan para mostrar sus partidas anteriores y prevenir trampas.

Anonimizado junto con la eliminación de cuenta.

Eliminado cuando ambos jugadores son anónimos.
