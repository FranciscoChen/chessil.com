隐私政策

ChessiL.com 是一个国际象棋网站。

ChessiL.com 将仅收集和使用提供服务所需的数据。

您可以创建一个帐户来玩排位棋赛并保留对局历史记录。

帐户和所有相关数据将在一年不活动后被删除。

您的数据不会与第三方共享。
